# Watchedlist project instructions

## Product
Watchedlist is a mobile-first personal movie history app focused on watched movies, simple ratings, and friends' taste.

## Design
- Mobile-first
- Calm, minimal, spacious UI
- Use Overpass font
- Accent color: #792E9C
- Use lots of whitespace
- Avoid heavy dividers and shadows
- Keep UI focused and not feature-heavy

## Technical
- Keep Supabase integration intact
- Use Next.js App Router
- Do not expose secret API keys to the client
- Keep TMDB calls server-side
- Preserve existing styling unless explicitly asked to redesign

## Product constraints
- No feed
- No comments
- No tags for now
- No algorithmic recommendations
- No unnecessary metadata
